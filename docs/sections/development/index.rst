Development Setup
-----------------

.. code-block:: bash

   # Clone repository
   git clone https://github.com/Mirjan-Ali-Sha/open-geodata-api.git
   cd open-geodata-api
   
   # Install in development mode
   pip install -e .[dev]
   
   # Run tests
   pytest
   
   # Check code style
   black .
   flake8
